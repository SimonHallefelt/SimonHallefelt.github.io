PORTFOLIO SUMMARY

Service Robotics

Built, designed and programmed a robot that is capable of to navigating a labyrinth, locate objects within it (cylinders), and carry them out of the labyrinth while meeting time constraints.


The robot’s brain was an Arduino programmed in C++. Ultrasonic and reflectivity sensors were utilized for navigation. The wooden components were designed using SolidWorks and cutout using a laser cutter.

SQL database

Designed and programmed an SQL database for a fictional cookie factory. The project involved creating a robust system to manage various aspects of the factory, including multiple recipes, ingredients, customers, and transportation logistics. Additionally, SQL queries were developed to automate key processes, such as checking inventory levels to ensure sufficient ingredients for fulfilling orders, among other functionalities.

Othello

This is an Othello game built with C++, featuring Raylib for its graphical user interface. The game supports three distinct types of players: a manual player, a random player, and a smart player that uses alpha-beta pruning to determine the best move.
The project is built using CMake and utilizes vcpkg to manage the Raylib library. Both the GUI and game logic run on separate threads, sharing the GameInfo class through a smart pointer (shared_ptr).

Connect-4

This is a Connect-4 game featuring a graphical user interface (GUI) and five player types: human, random, bot, bit bot (easy), and bit bot (hard). The game is developed in Rust and utilizes Slint to create the GUI. The bit bot (hard) employs both alpha-beta pruning and bit boards to determine the best move.
